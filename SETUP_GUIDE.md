# 設置與運行指南

## 1. 查看本機 IP

### macOS / Linux
```bash
ifconfig
```
尋找 WiFi 連線對應的 IP（通常形如 192.168.x.x）

### Windows
```cmd
ipconfig
```
尋找 WiFi 連線對應的 IPv4 位址

## 2. 單人版測試

### 步驟 1: 啟動伺服器（設備1）
```bash
python3 server.py
```

預期輸出：
```
伺服器已啟動: 192.168.x.x:5555
等待客戶端連線...
```

**記住 IP 位址！**

### 步驟 2: 啟動客戶端（設備2）
```bash
python3 client.py
```

**按提示輸入：**
1. 伺服器 IP：輸入上一步的 IP（例如 192.168.1.100）
2. 伺服器埠號：直接按 Enter（預設 5555）
3. 學號：輸入您的學號（例如 P76141259）

預期輸出：
```
已連線到 192.168.1.100:5555
Welcome Game P76141259
```

### 步驟 3: 進行猜拳
輸入選擇：rock、paper 或 scissors

不斷輸入直到贏得 3 次為止。

## 3. 多人版測試

### 步驟 1: 啟動多人版伺服器（設備1）
```bash
python3 server_multiuser.py
```

預期輸出：
```
多人版伺服器已啟動: 192.168.x.x:5556
等待2個客戶端連線...
```

**注意埠號是 5556！**

### 步驟 2: 啟動第1個客戶端（設備2）
```bash
python3 client.py
```

輸入：
- IP：伺服器 IP
- 埠號：5556
- 學號：任意用戶名

### 步驟 3: 啟動第2個客戶端（設備3）
```bash
python3 client.py
```

輸入同上

伺服器會自動進行 3 回合猜拳，然後顯示計分板。

## ⚠️ 常見問題

### "Connection refused"
檢查：
1. 伺服器是否仍在執行
2. IP 和埠號是否正確
3. 防火牆是否支援連線

### 無法連線
檢查：
1. 兩台設備是否在同一 WiFi
2. 防火牆設置

### 埠號被佔用
等待 1-2 分鐘後重試 或修改埠號行：
```python
server_port = 5555  # 改成其他號，如 5557
```
